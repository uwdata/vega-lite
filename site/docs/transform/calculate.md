The formula transform extends data objects with new fields (columns) according to an [expression](types.html#expression).

{: .suppress-error}
```json
{
  ...
  "transform": [
    {"calculate": ..., "as" ...} // Calculate Transform
     ...
  ],
  ...
}
```
