The `callout` composite mark represents a annotation with a line and a label.

To create a callout, we can set `mark` to `"callout"`:

{: .suppress-error}
```json
{
  ...
  "mark": "callout",
  ...
}
```

## Customizing Callout
To customize callout, we can set callout's mark definition object.
{: .suppress-error}
```json
{
  ...
  "mark": {
    "type": "callout",
    "angle": ...,
    "lineOffset": ...,
    "lineLength": ...,
    "labelOffset": ...
  }
  ...
}
```
